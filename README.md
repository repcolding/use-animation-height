# use animation height

Анимированное изменение высоты, через ```css transition```,
с установкой в ```auto```

***Локальный модуль verno.digital***

## Установка

```shell
# Через npm
npm i @verno.digital/use-animation-height

# Через yarn
yarn add @verno.digital/use-animation-height
```
